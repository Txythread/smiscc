**smiscc - smisc Compiler**

Compiles code (no llvm).

**Options:**
* Arguments that are not flags and do not belong to any will be treated as input files.
* `-h` or `-help` prints this screen or other help screens if accompanied by the flags listed above.
* `--ol` lets you specify which optimizations should take place in a general level
* `--optimizations` lets you specify specific optimizations
* `--show-tokens` outputs tokens during compilation phase
